### 2.2	Teknik Pengujian
Teknik pengujian yang digunakan dalam Sistem Pengaduan Masyarakat disesuaikan dengan karakteristik sistem yang berfokus pada fungsionalitas dan interaksi pengguna.

Pengujian dilakukan menggunakan beberapa teknik, yaitu Black Box Testing, White Box Testing, dan Grey Box Testing. Black Box Testing digunakan untuk menguji fungsi sistem berdasarkan input dan output tanpa melihat struktur internal kode, seperti pengujian login, pengajuan pengaduan, serta penampilan status laporan. White Box Testing digunakan untuk menguji logika program dan memastikan bahwa setiap alur dalam sistem berjalan dengan benar, terutama pada proses validasi dan perubahan status pengaduan. Sementara itu, Grey Box Testing digunakan sebagai kombinasi untuk menguji sistem dengan pemahaman sebagian terhadap struktur internal, sehingga dapat meningkatkan efektivitas pengujian terutama pada proses integrasi antar fitur.

Pemilihan teknik ini bertujuan agar pengujian dapat dilakukan secara menyeluruh, baik dari sisi fungsional maupun dari sisi logika sistem, sehingga dapat meminimalisir kesalahan serta meningkatkan kualitas perangkat lunak sebelum digunakan oleh pengguna.
