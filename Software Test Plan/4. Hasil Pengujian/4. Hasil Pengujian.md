
# 4. Hasil Pengujian (Test Execution Report)

### 4.1 Hasil Pengujian Black Box
| ID Test | Skenario Pengujian | Status Akhir | Keterangan / Catatan |
|---|---|---|---|
| **BB-01** | Mengirim pengaduan dengan data valid | **PASS** | Sistem memunculkan notifikasi sukses dengan benar. |
| **BB-02** | Mengirim pengaduan dengan isi laporan kosong | **FAIL** | **Bug:** Sistem tetap memproses laporan masuk. Akan dilaporkan via GitHub Issues. |
| **BB-03** | Admin memfilter Laporan Bulanan | **PASS** | Data terfilter dengan benar sesuai bulan dan tahun. |
| **BB-04** | Admin memberi tanggapan kosong | **PASS** | Sistem menolak input dan memunculkan error validasi. |
