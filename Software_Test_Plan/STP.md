# SOFTWARE TEST PLAN

## 1. Pendahuluan

### 1.1 Tujuan
Dokumen ini dibuat untuk menjelaskan proses pengujian sistem.

1.2 Latar Belakang
Pengujian dilakukan untuk memastikan sistem berjalan dengan baik.

1.3 Ruang Lingkup
Pengujian meliputi seluruh fitur aplikasi.

2. Lingkungan Pengujian

2.1 Material Pengujian
- Source code
- Database

2.2 Teknik Pengujian
- Black Box
- White Box

3. Rencana Pengujian
Pengujian dilakukan berdasarkan test case.

## 4. Hasil Pengujian
Berisi hasil berhasil atau gagal.
